<h4>From the terminal, run the following commands:<br>
<br>
 $ sudo su<br>
 $ apt install gcc make<br>
 $ mkdir --parents /media/cdrom<br>
 $ mount /dev/cdrom /media/cdrom<br>
 $ /media/cdrom/VBoxLinuxAdditions.run<br>
 $ reboot</h4>
